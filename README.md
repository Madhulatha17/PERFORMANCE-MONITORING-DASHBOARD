# PERFORMANCE-MONITORING-DASHBOARD


## Overview

A dashboard for monitoring .NET application performance using Azure Application Insights.

## Features

- Request Monitoring
- Response Time Tracking
- Exception Monitoring
- Dependency Tracking
- Dashboard Visualization

## Technologies

- ASP.NET Core MVC
- Azure Application Insights
- .NET 8

## Run

```bash
dotnet restore
dotnet run
```

Open:

https://localhost:5001
